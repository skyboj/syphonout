## Key Technical Notes
- metal 0.29 API: `set_fragment_bytes(index, length, bytes)` — index FIRST
- objc2 ↔ metal interop: cast `device.as_ptr()` → `*mut AnyObject` for `msg_send!`
- IOSurface extraction: `ivar_getOffset` + raw byte arithmetic (not `object_getIvar` — CF type)
- CGLContext required for SyphonClient; shared single context across displays, locked with CGLLockContext
- Syphon.framework x86_64 only → runs under Rosetta; Metal works in same process
- Shaders compiled by Xcode → `default.metallib` → loaded via `device.new_default_library()`
